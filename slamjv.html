O que é ser humano

O que é ser humano Será que é amar? Será que é pensar Será que falar que somos diferentes é um engano

O que é ser humano O que é ter dignidade? Será que ter seus direitos garantidos é muita audacidade? Para Se deferir os direitos,é preciso saber o que é ser humano

O que é ser humano Ter liberdade de pensar? Ter liberdade de se expressar? É ter liberdade para se calar É reconhecer o certo e errado

O que é ser humano Será que é ser feliz? Ou continuar com desrespeito e coisas infantis? Nos mostra o que é ser desumano

O que ser desumano Será que é odiar? Será que é agir e desprezar o pensar? É velejar sem se preocupar com o resto do oceano
